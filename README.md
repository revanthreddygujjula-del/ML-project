# Predicting High-Performance Marketing Campaigns Using Supervised Machine Learning

## Project Overview

This project develops supervised machine learning models to predict high-performing marketing campaigns using customer and marketing-related features.

The project compares multiple machine learning models and evaluates their predictive performance using various evaluation metrics.

---

## Dataset

Dataset Source:
Kaggle Marketing Campaign Dataset

Dataset Link:
https://www.kaggle.com/

---

## Machine Learning Models Used

1. Random Forest
2. XGBoost
3. LightGBM

---

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
